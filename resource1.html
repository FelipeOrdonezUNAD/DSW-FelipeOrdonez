Uso de la propiedad Display

A continuación prestamos un video donde se muestra para que sirve y como se utiliza l apropiedad display.

En CSS podemnos crear clases que tendrán preconfigurados ciertos atributos de estitulo que podremos aplicar en cualquier momento dentro de nuestras páginas.

En el siguiente video podra observar que el diseñador WEB tiene un problema, ha creado los botones del menú de su pagina, pero estos se organizan automaticamente uno sobre el otros. Esto se debe a que predeterminamente la etiqueta div organiza la informacion así. Para solucionar esto, el diseñador web crea una clase llamada menu, donde usa la propiedad display para hacer que los botones se organicen horizontalmente de forma automatica.
